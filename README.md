# 📈 Sales Prediction using Python

## 📌 Project Overview

This project predicts product sales based on advertising expenditures on TV, Radio, and Newspaper using Machine Learning.

It was developed as part of the **CodeAlpha Data Science Internship**.

---

## 🎯 Objective

The goal of this project is to build a machine learning model that predicts sales based on advertising budgets.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

---

## 📂 Dataset

Dataset: Advertising.csv

Features:
- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget

Target:
- Sales

---

## 🤖 Machine Learning Algorithm

- Linear Regression

---

## 📊 Project Workflow

1. Import Libraries
2. Load Dataset
3. Data Cleaning
4. Data Visualization
5. Train-Test Split
6. Model Training
7. Model Evaluation
8. Sales Prediction

---

## 📈 Model Performance

Evaluation Metrics:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

The model achieved an excellent prediction accuracy with a high R² Score.

---

## 📷 Output

The project generates:

- TV vs Sales Graph
- Radio vs Sales Graph
- Newspaper vs Sales Graph
- Actual vs Predicted Sales Graph
